# Dark Souls RPG — Painel do Mestre

## Como rodar

ES modules exigem servidor HTTP (não funcionam via `file://`).

```bash
./servir.sh        # Linux/Mac
servir.bat         # Windows
```

Acessar:
- Mestre:    http://localhost:8000/painel_mestre.html?view=master
- Jogadores: http://localhost:8000/painel_mestre.html?view=players
